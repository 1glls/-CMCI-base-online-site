🚀 INSTALLATION
────────────────────────────────────────────────────────────────────────────────
$ ./deploiement/local/deploy-local.sh    # Installation automatique

Ou manuellement :
$ cd src/backend && npm install
$ cd src/frontend && npm install


🎯 DÉMARRAGE
────────────────────────────────────────────────────────────────────────────────
Terminal 1 - Backend :
$ cd src/backend
$ npm run dev                             # → http://localhost:5000

Terminal 2 - Frontend :
$ cd src/frontend  
$ npm run dev                             # → http://localhost:3000


🔐 CRÉER ADMIN
────────────────────────────────────────────────────────────────────────────────
$ ./otherthings/scripts/create-admin.sh

Ou avec curl :
$ curl -X POST http://localhost:5000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@cmci.be","password":"Pass123","name":"Admin"}'


🧪 TESTER L'API
────────────────────────────────────────────────────────────────────────────────
$ ./otherthings/scripts/test-api.sh

╔══════════════════════════════════════════════════════════════════════════════╗
║                          ⚙️ CONFIGURATION REQUISE                             ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. Copier le fichier de configuration :
   $ cd src/backend
   $ cp .env.example .env

2. Éditer .env avec vos informations :
   - DATABASE_URL          (PostgreSQL ou MySQL)
   - JWT_SECRET            (Secret sécurisé)
   - YOUTUBE_API_KEY       (Optionnel)
   - FACEBOOK_ACCESS_TOKEN (Optionnel)
   - INSTAGRAM_ACCESS_TOKEN (Optionnel)
   - TIKTOK_ACCESS_TOKEN   (Optionnel)

3. Initialiser la base de données :
   $ npm run prisma:generate
   $ npm run prisma:migrate

4. Créer les dossiers d'upload :
   $ mkdir -p uploads/{events,testimonials,gallery}
